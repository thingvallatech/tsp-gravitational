- Props: `algoId: string` (looks up result from store)

- Contains a small (200x200) canvas that draws the current tour step for this algorithm (reuse the drawing logic from TourOverlay2D but self-contained -- draw city dots + tour edges in the algo's color based on `algorithmResults[algoId]` and `playbackStep[algoId]` from store)

Create Leaderboard.tsx -- a sorted results table:
- Reads ALL `algorithmResults` from store via `useStore`
- Sorts entries by `tourLength` ascending (shortest = best)
- Computes optimality gap: `((tourLength - bestTourLength) / bestTourLength * 100).toFixed(1)` for each, where bestTourLength is the minimum
- Displays a table with columns: Rank, Algorithm Name (with color dot), Tour Length, Compute Time (ms), Gap (%)
- The #1 ranked row gets a subtle gold highlight (bg-yellow-900/20)
- If no results, show "Run algorithms to see rankings"
- Use Tailwind for a clean dark table: text-sm, border-collapse, alternating row backgrounds
  </action>
